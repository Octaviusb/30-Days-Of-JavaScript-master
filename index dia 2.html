//¿Qué es React?

React es una biblioteca de JavaScript para crear una interfaz de usuario (UI) reutilizable.

//¿Qué es una biblioteca?

React es sí misma es una biblioteca que provee los elementos para crear la interfaz de usuario

//¿Qué es una aplicación de una sola página?

Una aplicación de página única (SPA) es esencialmente una página web que interactúa dinámicamente con el navegador web al reescribir la página web actual con los datos obtenidos del servidor web.

//¿Qué es un componente?

son funciones de JavaScript que aceptan entradas arbitrarias (llamadas “props”) y retornan elementos de React que describen lo que debe aparecer en la pantalla.

//¿Cuál es la última versión de React?

la ultima version de react es la 18.2.0

//¿Qué es DOM?

es la estructura en el navegador de la pagina web

//¿Qué es react DOM virtual?

El DOM virtual (VDOM) es un concepto de programación donde una representación ideal o “virtual” de la IU se mantiene en memoria y en sincronía con el DOM “real”

//¿Qué tiene una aplicación web o un sitio web (compuesto por)?

Las aplicaciones web tienen una arquitectura cliente-servidor. Su código se divide en dos componentes: scripts del lado del cliente y scripts del lado del servidor.

//¿Por qué react?

porque permite desarrollar interfaces de usuarios mediante componentes. Estos componentes pueden ser tanto, cada una de las piezas que forman la interfaz, como la propia interfaz completa. Cada componente contiene tanto la lógica como la parte visual, de este modo podemos reutilizarlo dentro de otros componentes.

//Ejercicios: JSX //¿Qué es un elemento HTML?

JSX nos permite escribir elementos HTML con código JavaScript. Un elemento HTML tiene etiquetas de apertura y cierre, contenido y atributo en la etiqueta de apertura.

//¿Cómo escribir un elemento HTML de cierre automático?

Los elementos JSX pueden tener elementos secundarios, al igual que HTML. Sin embargo, si un elemento no tiene hijos, debe cerrarse con una barra diagonal ( /).

//¿Qué es un atributo HTML? Escribe algunos de ellos

son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios, Ejemplo: class, border, cols, colspan.

//¿Qué es JSX?

JSX significa JavaScript XML. JSX nos permite escribir elementos HTML con código JavaScript.

//¿Qué es Bábel?

Es el transpilador que convierte el codigo jsx a js puro

//¿Qué es un transpilador?

Transpilador es un tipo especial de compilador que traduce de un lenguaje fuente a otro fuente también de un nivel de abstracción parecido.
